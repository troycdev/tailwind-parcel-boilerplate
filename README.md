# Tailwind Parcel Boilerplate

### Description

Basic boilerplate project for getting started with Tailwind projects using Parcel. Supports Sass, with autoprefixer and purge support for unused styles.

### Usage

To get started, clone this repository and run `npm install` to install all required dependencies. Spin up a development server with `npm run dev`, and when you're ready to build use `npm run build`.

### Built With

- [Tailwind CSS](https://tailwindcss.com/) - CSS framework
- [Parcel](https://parceljs.org/) - Application bundler
- [Sass](https://sass-lang.com/) - CSS preprocessor
- [PostCSS](https://postcss.org/) - CSS automation
- [Autoprefixer](https://www.npmjs.com/package/autoprefixer) - Vendor prefix tool
